# 数组中的K-diff数对

## [题干](https://leetcode-cn.com/problems/k-diff-pairs-in-an-array/)

给定一个整数数组和一个整数**k**, 需要在数组中找到**不同的** `k-diff` 数对，这里将 **k-diff** 数对定义为一个整数对 (i,j), 其中 `i` 和 `j` 都是数组中的数字，且两数之差的绝对值是 `k`.

注意：

- 数对 (i,j) 和数对 (j,i) 被算作同一数对
- 数组的长度不超过 10000
- 所有输入的数组的范围在 [-1e7,1e7]

难度：简单
